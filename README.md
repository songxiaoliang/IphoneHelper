# IphoneHelper
Iphone 机型适配工具类

### 使用方式

##### 1. 引入
```javascript
import { isIphoneX, isIphoneXS, isIphoneXR, isIphoneXSMax, isXSeriesIphone, isPlusSizeIphone } from 'IphoneHelper';
```
##### 2. 使用
```javascript
isIphoneX();
isIphoneXS();
isIphoneXR();
... ...
```
### 注:
isPlusSizeIphone(): Iphone 6, 7, 8 (包含plus)
isXSeriesIphone(): Iphone X 系列(包含 X, XR, XS, XSMax)
